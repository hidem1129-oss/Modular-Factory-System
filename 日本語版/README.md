# Modular Factory System

> 分散型I²Cノード、実機ハードウェア、データ駆動型オーケストレーションで構築された、再構成可能なDX（デジタルトランスフォーメーション）プラットフォーム

## このプロジェクトについて

本プロジェクトは、以下を統合した  
**実機ベースのDX（デジタルトランスフォーメーション）検証環境**です。

- 制御（マイコン）
- アクチュエータ（モーター等）
- センサ（フォトリフレクタ等）
- データ収集・可視化（Grafana等）

これらを**モジュール化し、自由に組み換え可能**にすることで、

- 物流ライン
- 製造ライン
- AI検査工程

といった様々なシナリオを再現できます。

---

## 背景、課題
実際のDX案件では、

- 現場（ハードウェア）
- 制御（組み込み）
- データ（IT）

が分断されているケースが多く、「全体像が見えにくい」、「PoCが机上で終わってしまう」ことがあります。

## このプロジェクトで解決すること

本プロジェクトでは、

- 実機を用いた再現
- 制御とデータの統合
- モジュール化による再利用

により、

**DXを「動く形」で可視化すること**を目的としています。
本プロジェクトは、

- ハードウェアとソフトウェアの統合
- データ可視化
- 柔軟なシステム構成

を実機で再現することで、**DXの本質（現場×データ×制御）を可視化する試み**です。

## アーキテクチャ
```text
[ Raspberry Pi 5 ]
        │ I²C
 ┌──────┴─────────┐────────────────┐────────────...
 │                │                │
Servo Node   Sensor Node       Motor Node       ...
 │                │                │
Actuator      Measurement       Actuator        ...
```
- 上位制御：Raspberry Pi 5
- ローカル制御：Raspberry Pi Pico
- 通信：I²C
- 構成：分散型ノード構造
