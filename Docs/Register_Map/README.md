# Register Map

This directory contains reference documents for the register-based I²C protocol used in the Modular Factory System.

It includes:

- register addresses
- status bit definitions
- command bit definitions
- feedback / setpoint register mapping
- bit-pattern mapping documents

## Files

| File | Description |
|---|---|
| `bit_pattern_mapping.xlsx` | Spreadsheet reference for register and bit-pattern mapping |
