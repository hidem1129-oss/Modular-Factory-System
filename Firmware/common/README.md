# common

Shared firmware components used across all Raspberry Pi Pico nodes.

## Responsibilities

- Core node behavior
- Register-based protocol handling
- Platform abstraction
- Public shared headers
- Common design documentation

## Directory Structure

- `core/` : shared node logic and state handling
- `protocol/` : register map and protocol definitions
- `platform/` : hardware/platform-dependent implementation
- `include/` : public shared headers
- `docs/` : common design notes
