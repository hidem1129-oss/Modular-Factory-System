# Grafana

This directory contains notes for visualizing the SQLite logs generated by the I2C Debugger.

Grafana is used as a visualization layer, not as a hardware control layer.

---

## Purpose

Grafana displays:

- node state timelines
- recent events
- error and warning logs
- monitoring sessions
- power monitor values
- current / voltage / power trends

---

## Data Source

The dashboard uses SQLite as its data source.

The database is generated by the monitoring software: `/var/lib/i2c_debugger/i2c_debugger_events.sqlite3`

---

## Notes

Most Grafana-related documentation in this project consists of:

- SQL queries
- dashboard screenshots
- panel configuration notes
- time-range and session-filter notes
